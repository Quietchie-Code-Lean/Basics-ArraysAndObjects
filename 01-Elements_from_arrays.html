Avoiding unnecessary DOM updates

Each time we assign a new value to the innerHTML attribute, we trigger a page refresh. However, we can avoid this refresh by using a new variable to store the information. After completing the loop, we can perform a single page refresh using the content stored in the variable. This optimizes page performance.
